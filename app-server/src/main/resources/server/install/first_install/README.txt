Здесь лежат файлы для установки сервиса приложений, именно первоначальной установке:
	- app-manager.ico - иконка для сервиса приложений
	- vbscript.vbs - скрипт для создания иконки на рабочем столе
	- app_service_installer.zip - архив с запускатором и обновлятором сервиса приложений
		- в корне архива должен лежать только файл: updater_app_service.jar
