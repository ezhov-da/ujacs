Здесь лежат настройки для всех приложений:
	- установщика
	- запускатора
	- каждого приложения

Файл apps.xml:
	содержится список приложений, которые отражаются в сервисе приложений, 
	для добавления приложения нужно редактировать именно этот файл

Файл config.properties:
	основной файл в котором содержатся настройки для всех взаимодействующих с сервером приложений,
	то есть при запуске такого приложения, происходит загрузка данного файла
